Circles

If we have (x-3)2 + (y+4)2 = 20 - then we would be able to find the centre of the circle by using the negatives of the numbers after x and y so the centre's coordinates are (3,-4).

If we wanted to prove that the circle passes through the point (5,-8) then we could just substitute in x=5 and y=-8 int othe equation.

(5-3)2 + (-8 + 4)2 = 4 + 16, which equals 20, therefore, it works with the equation and the circle passes through the point.
